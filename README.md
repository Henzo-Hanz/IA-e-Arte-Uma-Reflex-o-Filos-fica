# IA-e-Arte-Uma-Reflex-o-Filos-fica
O trabalho discute a relação entre inteligência artificial e arte, refletindo sobre criatividade, autoria e estética. Analisa se obras geradas por algoritmos podem ser consideradas arte, os impactos culturais dessa produção e os dilemas éticos envolvidos, propondo uma visão filosófica sobre esse fenômeno
